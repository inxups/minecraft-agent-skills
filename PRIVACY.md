# Privacy

`minecraft-agent-skills` is a public skills and plugin-bundle repository. The repo itself does not collect data, run telemetry, or operate a hosted backend.

It contains Markdown skills, helper scripts, fixtures, docs, and plugin manifests. GitHub Actions may run repository checks on commits and pull requests, using the data already present in the repository.

Using these skills with Codex, Claude Code, GitHub, OpenAI APIs, Minecraft services, or other tools is governed by those services and by your local configuration. This repository does not change their privacy behavior.

Do not put secrets, API keys, tokens, private server data, or sensitive project content in public issues, pull requests, examples, or fixtures.
